# Pokedex CLI

### Description
Psuedo-Pokemon game in a command-line REPL. It uses the [PokeApi](https://pokeapi.co/) 

### Functionality 
* Explore regions
* check pokemon in that region
* Catch pokemon and name it
* Inspect your pokemon
