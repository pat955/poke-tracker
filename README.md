# Pokedex CLI

### Description
Psuedo-Pokemon game in a command-line REPL. It uses [PokeApi](https://pokeapi.co/) for info.

### Functionality 
* Explore areas
* Check pokemon in that region
* Catch pokemon and name them
* Inspect your pokemon
* Buy items and check inventory
